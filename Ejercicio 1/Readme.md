# 📚 Ejercicio de Programación Orientada a Objetos (POO)
## Sistema de Calificación de Estudiantes

### 📌 Descripción

Este ejercicio tiene como objetivo aplicar los principios de **Programación Orientada a Objetos (POO)** en Python mediante la creación de un sistema simple que permita registrar información de un estudiante y mostrar su calificación.

El programa utiliza **clases, atributos, métodos y herencia** para organizar el código de manera clara y reutilizable.

---

## 🎯 Objetivo

Crear un programa que:

- Defina una **clase base llamada `Estudiante`**
- Permita guardar información del estudiante
- Calcula o muestre su calificación
- Utilice los principios básicos de la **POO**
