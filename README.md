# 📌 Introducción

Este repositorio forma parte de mi formación en la carrera de **Ingeniería en Sistemas**, donde se estudian y aplican diferentes conceptos fundamentales de la programación y el desarrollo de software.

En esta materia se trabaja con **Programación Orientada a Objetos (POO)**, un paradigma de programación que permite organizar el código de forma más estructurada, reutilizable y fácil de mantener. A través de diferentes ejercicios y proyectos prácticos, se busca comprender cómo modelar problemas del mundo real utilizando **clases, objetos, atributos y métodos**.

---

# 🎯 Objetivo del repositorio

El objetivo de este repositorio es reunir distintos **ejercicios y prácticas realizadas durante la materia**, en los cuales se aplican los principios de la Programación Orientada a Objetos utilizando **Python**.

Cada ejercicio tiene como propósito fortalecer la comprensión de conceptos como:

- Creación de **clases y objetos**
- Uso de **atributos y métodos**
- **Encapsulación**
- **Herencia**
- **Polimorfismo**
- Organización y reutilización del código

# 🧠 Importancia de la Programación Orientada a Objetos

La **POO** es uno de los paradigmas más utilizados en el desarrollo de software moderno. Permite:

- Organizar programas complejos en partes más pequeñas
- Reutilizar código de manera eficiente
- Facilitar el mantenimiento y la escalabilidad de los sistemas
- Representar entidades del mundo real mediante objetos dentro del programa

Gracias a estos beneficios, la Programación Orientada a Objetos es ampliamente utilizada en lenguajes como **Python, Java, C++, C# y muchos otros**.
