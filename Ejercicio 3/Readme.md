# 🐾 Ejercicio de Programación Orientada a Objetos (POO)
## Sistema de Animales con Herencia

---

# 📌 Descripción

Este ejercicio tiene como objetivo aplicar el concepto de **herencia en Programación Orientada a Objetos (POO)** utilizando clase base y sus hijas
---

# 🎯 Objetivo

El programa permite:

- Crear una clase base `Animal`
- Crear clases hijas `Perro` y `Gato`
- Hacer que los animales se presenten
- Hacer que cada animal emita su sonido

---

# 📁 Estructura del Proyecto
sistema_animales/
│
├── Animal.py
├── Perro.py
├── Gato.py
└── main.py


📌#CONCLUSION 

Este ejercicio muestra cómo la Programación Orientada a Objetos permite modelar sistemas del mundo real, como una cuenta bancaria, mediante clases y métodos que representan sus operaciones principales.
