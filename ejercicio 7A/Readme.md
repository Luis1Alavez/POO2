# ⚠️ Proyecto: Excepciones en Python

## 📖 Descripción

En esta carpeta se desarrollan **5 ejercicios prácticos sobre manejo de excepciones en Python**, que van desde nivel básico hasta nivel difícil.  
El objetivo es aprender a:

- Detectar errores en tiempo de ejecución.  
- Usar bloques `try` / `except` para controlar excepciones.  
- Manejar múltiples tipos de excepciones.  
- Aplicar `finally` y `else` cuando sea necesario.  
- Crear excepciones personalizadas (`raise`).  

Cada ejercicio está pensado para reforzar conceptos clave de **manejo seguro de errores** en Python y fomentar buenas prácticas de programación.
