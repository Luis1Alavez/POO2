# 🍽️ Ejercicio de Programación Orientada a Objetos (POO)
## Sistema de Menú de Restaurante

---

# 📌 Descripción

Este ejercicio consiste en desarrollar un sistema simple de **menú de restaurante** utilizando **Programación Orientada a Objetos (POO)** en Python.

El programa permite representar diferentes platillos del menú y mostrar su información, como el nombre del platillo y su precio.

El objetivo es aplicar conceptos fundamentales de la **POO**, como clases, atributos y métodos.

---

# 🎯 Objetivo

Crear un programa que permita:

- Representar los platillos de un restaurante
- Mostrar el nombre del platillo
- Mostrar el precio del platillo
- Organizar el código utilizando **clases y objetos**
